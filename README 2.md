# Atlas Level 2 — Opportunity Hunter And Analyst

**Owner: Mazen Farkouh**

Atlas scans public GitHub issues and selected public Algora bounty boards, extracts visible rewards, applies legal/safety filters, scores opportunities, and publishes a live dashboard.

## New Level 2 capabilities

- Runs every 15 minutes through GitHub Actions
- Searches multiple opportunity queries
- Scores reward, freshness, scope, and competition
- Rejects blocked legal/safety terms
- Creates `PURSUE`, `REVIEW`, and `REJECT` decisions
- Publishes a mobile-friendly live dashboard
- Costs $0 in a public GitHub repository

## It does not yet

- Claim bounties
- Write code autonomously in the background
- Submit pull requests
- Guarantee payment

External actions remain human-approved.
