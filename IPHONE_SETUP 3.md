# iPhone Setup

1. In GitHub, create a **public** repository named `atlas-opportunity-hunter`.
2. Download and unzip this package in Files.
3. Upload all files and folders, keeping names unchanged.
4. Open the repository's **Actions** tab.
5. Open **Atlas Opportunity Scan** and press **Run workflow**.
6. Open **Settings → Pages**.
7. Select **Deploy from a branch**, branch `main`, folder `/docs`.
8. Save. GitHub will display the Atlas dashboard link.

Do not enter PayPal, bank, tax, password, private-key, or identification information into GitHub.
